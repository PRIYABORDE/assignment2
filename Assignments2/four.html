List of Tags.

  1. h1: Defines a heading level 1.
  2. br: Defines a break in line
  3. A: tag Defines a hyperlinks
  4. p: Defines a paragraph of text.
  5. div: Defines a container for HTML elements.
  6. table: Defines a table with rows and columns.
  7. ul: Defines an unordered list.
  8. title: Page Title
  9. href: attribute of specifies the URL of the page the link goes to
  10. src: attribute of specifies the path to the image to be displayed
  11. width and height: attributes of provide size information for images
  12. alt attribute of provides an alternate text for an image
  13. style: attribute is used to add styles to an element, such as color, font, size, and more
  14. lang: attribute of the tag declares the language of the Web page
